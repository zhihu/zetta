# Zetta - TableStore On TiKV

## Introduction

Zetta is an open-source NewSQL database that supports Hybrid Transactional and Cloud Spanner like API 


## Quick start

```bash
$> cd zetta
$> make zetta-server
$> ./bin/zetta-server -store tikv -path='{pd.address:port}'
```
